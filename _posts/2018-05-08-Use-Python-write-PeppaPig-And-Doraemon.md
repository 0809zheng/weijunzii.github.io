#  0  前言

***

最近发现了很有意思的东西，有人用 turtle 这个库画了一个小猪佩奇，跑去 GitHub 看了一下代码，发现写这个的人真的很有耐心，居然能写下来。

再找了找，发现有人还画了哆啦 A 梦， GitHub 果然人才多。
